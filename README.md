# CV-Aldohdezco

Portafolio y CV bilingüe construido con Next.js.

## Desarrollo

1. Instala Node.js 20 o superior.
2. Ejecuta `npm install` dentro de `CV-Aldohdezco`.
3. Arranca con `npm run dev`.

## Rutas útiles

- `/` portada principal bilingüe.
- `/cv/es` CV en español.
- `/cv/en` CV en inglés.
- `/cv/es/download` descarga la versión en español.
- `/cv/en/download` descarga la versión en inglés.

Los archivos `index.html` y `styles.css` quedan como referencia del CV estático original.
